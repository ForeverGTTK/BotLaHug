Welcome to BotLaHug

BotLaHug is a comprehensive club management system designed to allow multiple clubs to register, manage athletes, organize seasons, and maintain their own customized webpages. The platform simplifies the process of managing athlete and club data, offering an easy-to-use interface for both administrators and club managers.

Key Features

BotLaHug offers the following essential functionalities:

  • Club Registration: Clubs can sign up and manage their own unique webpage, complete with custom details and media.
  • Athlete Management: Clubs can register and manage athletes, track their participation, and store relevant data.
  • Season Management: Clubs can create and manage seasons, associating athletes with different seasons for competitions and events.
  • Basic Payment System: For demonstration purposes, a made-up payment system is integrated, providing placeholders for more advanced features.

What's Next?

Future developments will expand on BotLaHug’s capabilities, including:

  • Customized Designs for Clubs: Each club will be able to further personalize its website design, including color schemes and layout adjustments.
  • Competition Management: Additional models and features to manage competitions and track results.
  • Payment Integration: Adding support for various online payment methods, including קבלות ואפשרויות סליקת אשראי ותשלום מקוון (credit card processing and online payment receipts).

Stay tuned for updates and new features that will enhance the functionality and flexibility of the platform!

Getting Started

To get started with BotLaHug, follow these steps to download the repository and set up the application on your local machine.

Steps to Get Started

  1. Install Git (if you don't have it installed):

    Download and install Git from https://git-scm.com. Follow the setup instructions for your operating system.

  2. Clone the repository from GitHub:

    Open a terminal or command prompt, navigate to the folder where you want to store the project, and run:

    git clone https://github.com/username/botlahug.git

    Replace username with the actual GitHub username. This command will download the project files from the repository.

  3. Navigate to the project directory:

    Change into the project’s directory using:

    cd botlahug

  4. Install the dependencies:

    For Python projects, use pip to install the required packages by running:

    pip install -r requirements.txt

  5. Run the development server:

    Start the local server with the following command:

    python manage.py runserver

    Open your browser and navigate to http://127.0.0.1:8000 to explore the application.
