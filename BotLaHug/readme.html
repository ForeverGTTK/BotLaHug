Welcome to BotLaHug

BotLaHug is an innovative club management system designed to cater to the unique needs of multiple clubs. It allows clubs to register and create custom webpages, manage their athletes, schedule classes, and track seasonal participation. With a user-friendly interface, BotLaHug streamlines club management, helping administrators and managers save time and focus on what matters most — the club members and activities.

Key Features

  • Multi-Club Registration: Each club can create and manage its own personalized webpage, complete with unique details, images, and design elements.
  • Athlete & Class Management: Clubs can register athletes, organize classes, and manage athletes' participation in classes. They can also keep records of athlete details such as contact info, year of joining, and profile pictures.
  • Season Tracking: Clubs can create seasons, enroll athletes in specific seasons, and monitor their engagement.
  • Integrated Payment System: Although a basic placeholder for now, BotLaHug includes a payment system that can be extended to integrate real-world payment gateways and manage financial records for athlete participation.

What’s Next?

The following features are on the roadmap for BotLaHug:

  • Club-Specific Customizations: Clubs will be able to fully customize their webpage design, including colors, fonts, and layouts, offering them a branded and unique online presence.
  • Advanced Competition Management: New tools will be added for clubs to create and manage competitions, track scores, and monitor athlete performance in real-time.
  • Enhanced Payment System: BotLaHug will support various online payment methods, including integration for credit card processing, and the option to generate receipts for payments.

Getting Started

Follow the steps below to set up BotLaHug on your local machine and start managing clubs right away.

Steps to Get Started

  1. Install Git: Download and install Git from https://git-scm.com. Follow the installation guide for your operating system.
  2. Clone the BotLaHug repository: Open a terminal or command prompt, navigate to the folder where you want to store the project, and run:
    git clone https://github.com/username/botlahug.git
  3. Navigate to the project directory: Use the command cd botlahug to enter the project folder.
  4. Install dependencies: Run pip install -r requirements.txt to install the necessary Python packages.
  5. Start the development server: Run python manage.py runserver to start the server. You can then open a browser and go to http://127.0.0.1:8000 to start exploring the app.
